## 2022-02-02

### upper-beginner-2-sending-a-package-in-greece

* το πακέτο - package, packet
* μέσα - inside
* το δώρο - gift, present
* κοστίζω - to cost
* τα ρέστα - change
* πότε - when
* η εβδομάδα - week
* δεσποινίς - miss
* υπογράφω - to sign

καλημέρα σας - more formal greeting, good day to you sir / madam.

τ, δ, θ, ζ, and ν become σ.

For Example:

ντύνω -> ντύσω
γυρίζω -> γυρίσω

κ, γ, ζ, χ, and χν become ξ.

For Example:

βρέχει -> βρέξει
δείχνω -> δείξω
κοιτάζω -> κοιτάξω

π, β, φ, and πτ become ψ.

For Example:

γράφω -> γράψω
λείπω -> λείψω

If the word stem ends in a vowel, it is likely that we will need to add -s.

For Example:

ακούω -> ακούσω
ρωτώ -> ρωτήσω
μιλάω -> μιλήσω
συγχωρώ -> συγχωρήσω
